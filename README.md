## Descripción

Página web de la organización, usando: 

* [Next.js](https://nextjs.org/). 
* [Tailwindcss](https://tailwindcss.com/)

## Instalación 

1. **Clona** el proyecto. 

   ```bash
   git clone https://github.com/manuelalferez/petcan.git
   ```

2. **Instala** los paquetes.

   ```bash
   cd website 
   npm i
   ```

3. **Ejecuta** el proyecto.

   ```bash
   npm run dev
   ```
